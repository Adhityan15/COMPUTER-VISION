Computer Vision Lab
📌 Overview

This repository contains all Computer Vision laboratory experiments implemented using Python. The experiments focus on understanding how machines interpret and analyze visual data using image processing and feature extraction techniques.

The aim of this lab is to build foundational knowledge in computer vision concepts that are widely used in Artificial Intelligence, Machine Learning, and real-time vision systems.

🛠️ Technologies Used

Python 3.x

OpenCV

NumPy

Matplotlib

🧠 Concepts Covered

Image Representation and Processing

Feature Detection

Image Segmentation

Edge Detection

Thresholding

Object Detection Basics

Morphological Operations

Transformations
